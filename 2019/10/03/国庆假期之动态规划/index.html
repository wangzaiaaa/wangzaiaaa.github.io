dp描述

dp(dynamic programming) ，即动态规划，动态规划是解决算法问题的一种方法，其特点是最优子结构和重复子问题，我觉得dp和分治法有点类似，但是dp可以利用重复子问题，不需要重复求解。

Problem64

给定一个包含非负整数的 m x n 网格，请找出一条从左上角到右下角的路径，使得路径上的数字总和为最小。

说明：每次只能向下或者向右移动一步。

示例:

输入:
[
[1,3,1],
[1,5,1],
[4,2,1]
] 输出: 7
解释: 因为路径 1→3→1→1→1 的总和最小。

思路

假设从(0,0)到（i，j）的最小路径为f(i，j)，when j = 0 f(i，0) = f(i-1，0) + matrix(i，0)；when i = 0 f(0，j) = f(0，j-1) + matrix(0，j)； others f(i，j) = f(i，j) = matrix(i，j) + Min(f(i-1，j),f(i，j-1))
