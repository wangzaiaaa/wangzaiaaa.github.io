前言

本篇文章主要介绍linux文本编辑的内容，设置到vi和vim的一些操作。

编辑模式

vi的基本操作

启动vi，并向其传递一个不存在的文件名，就可以通过vi创建新文件。

如果想在文件中添加一些内容，可以按下i或者insert键进入编辑模式。

如果要保存我们的修改，我们可以在命令模式下输入一条ex命令，如何输入w

移动光标

在命令模式下，vi提供了很多移动光标的命令，如下：

键 光标操作
L或者右方向键 右移动一位
H或者左方向键 左移动一位
j或者下方向键 下移动一位
k或者上方向键 上移动一位
Shift-6(^) 移动到本行第一个非空字符
Shift-4($) 移动到本行的尾
W 移动到下一个单词或者标点的开头
Shift-W(W) y移动到下一个单词的开头，忽略标点
B 移动到上一个单词或者标点的开头
Shift-B(B) y移动到上一个单词的开头，忽略标点
Shift-G(G) 移动到文件的最后一行

基本编辑

添加文本

因为光标不能跳出行末，所有单纯的适用i命令不能完成在文本行末添加内容的任务。为此vi提供了在行末添加文本的a命令。当用户将光标移动到行末时并适用a命令，光标就会跨越文本的末尾，同时进入插入模式。

因为用户经常适用到在行末添加文本的功能，所以vi提供了使光标移动到行末并且进入插入模式的A命令

插入一行

插入文本的另一种方式是在文本中重开一行，即在现存的文字中插入空白行并进入插入模式。

命令 开行
o 在当前行的下方
O 在当前行的上方

删除文本

vi提供了很多删除文本的方式，没一种都需要进行一次到两两次的按键操作。首先，X键会删除光标处的字符。x命令可以添加数字前缀来明确删除的字符数量，除此以为还有d命令。

命令 删除内容
x 当前字符
3x 当前和之后的2个字符
dd 当前行
5dd 当前和之后的5行
dW 当前字符到下一个单词的开始
d$ 当前字符到当前行的末尾
d0 当前字符到当前行的起始
d^ 当前字符到当前行下一个非空白字符
dG 当前行到文件末尾
d20G 当前行到文件第20行

剪切，复制和黏贴文本

d命令并不是简单的删除文本，而是在剪切文本，每次用户在使用d命令之后，都会复制删除的内容进入缓存，然后用户就可以使用p命令将其缓存中的内容黏贴到光标之后会之前。

像d命令剪切文本的形式一样，命令y会复制文本。

命令 复制内容
yy 当前行
5yy 当前行和之后的4行
yW 当前字符到下一个单词的起始
y$ 当前字符到当前行的末尾
y0 当前字符到当前行的起始
y^ 当前字符到当前行下一个非空白字符
yG 当前字符到当前行尾
y20G 当前行到文件第20行

合并行

J命名可以将本行和下一行合并(注意与j区别

查找和替换

vi提供了在一行或者整个文件中，根据搜索条件将光标移动到指定位置的功能。vi还可以执行文本替换工作。

行内搜索

命令f可以在行内进行搜索，并将光标移动到搜索到的下一个指定字符。

搜索整个文件

/命令可以进入全文搜索模式，按下n进行下一次搜索

vi使用ex命令来执行几行之内或者整个文件中的搜索和替换操作

组成 含义
c 每次替换都会询问
: 分号用于启动一条ex命令
% 确定操作的范围，%表示从第一行到最后一行，1,5表示第一行到第五行
s 指定了具体的操作
/Line/line 搜索和替换的文本
g 指代全局，缺省下只会替换第一行符合条件的实例

编辑多个文件

可以通过上条命令打开多个文件，:n 切换到下一个文件，:N 切换到上一个文件
